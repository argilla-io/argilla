Web App UI

This section contains a quick, but throrough overview of Rubrix web-app's User Interface (UI).

---------
The web-app has two main sections: 
- `**Workspace** <workspace.rst>`_\ 
- `**Dataset** <dataset.rst>`_\

Features
---------

- **Explore Records**
- **View Metrics**
- **Filter Records**
- **Define Labeling Rules**

- **Sidebar**
